# IDEA EDU course project

## Project
This is the *Last Pencil (Java)* project that is part of Hyperskill platform from Jetbrains Academy.

Purpose of the project is to learn Java basics and to get more familiar with its ecosystem.

## Technologies

- Java 17

## Project description
Last Pencil is a fun program where you draw with the computer, each player trying to follow the defined winning strategy.

Player and bot each draw a number of pencils, between (1, 3) until there is none left. 

Whoever draws the final pencil loses.
